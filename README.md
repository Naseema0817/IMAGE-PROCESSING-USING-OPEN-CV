# IMAGE-PROCESSING-USING-OPEN-CV

This project demonstrates basic image processing techniques using OpenCV in Python, including:

- Reading and displaying an image
- Grayscale conversion
- Blurring
- Thresholding
- Canny edge detection
- Contour detection
- Pixel value access
- Histogram analysis
